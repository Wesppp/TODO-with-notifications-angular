About the web application

This application is designed for taking notes. Create a folder on a specific topic. For example, "Sports".
And make yourself a series of tasks before a certain deadline.
You can move tasks between the "TODO" "DOING" "DONE" columns.
"TODO" - if you haven't started working on this task yet
"DOING" - if you are actively working on the task
"DONE" - if you have completed the task
Once a day you will receive an email message about the deadlines of your tasks
